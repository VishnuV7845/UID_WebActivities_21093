Bugstars

Bugstars is a Debug the Code event. Particpants are given applications that have already been made, but the code is not 100% correct! There are multiple errors and bugs in the code. These are not revealed to the participants, so they would have to go through the code and debug it themselves. Scores are awarded based on the percentage of bugs fixed.
